mapIfSource

  • mapIfSource<I, T, F>(predicate: PredicateFn<I>, trueResult: MapFn<I, T>, falseResult: MapFn<I, T | F>): OperatorFunction<I, T | F>
    • Defined in libs/rxjs/utility/src/lib/map-if-source.ts:47

    The mapIfSource operator is used with an Observable value and takes a predicate function. Based on the result of the predicate it will return a value based on a truthy or falsy result. Each value can be cast to a specific type if required

    example
    of('dog').pipe(
 mapIfSource<string, string>(
   (value) => value === 'dog',
   (value) => 'woof! woof!',
   (value) => `meow!`,
 ),
).subscribe(console.log) // 'woof! woof!'
    example
    of('42').pipe(
 mapIfSource<string, number, string>(
   (value) => value === '42',
   (value) => parseInt(value),
   (value) => `${value}: This is not the ultimate answer`,
 ),
).subscribe(console.log) // 42

    Type parameters

    • I

      The value type for the incoming observable source

    • T

      The type returned from the Truthy result

    • F

      The type returned from the Falsy result, this type is optional and if not included the T type will be used

    Parameters

    • predicate: PredicateFn<I>
    • trueResult: MapFn<I, T>
    • falseResult: MapFn<I, T | F>

    Returns OperatorFunction<I, T | F>

    Any value based on the Truthy or Falsy MapFn based on the PredicateFn result
