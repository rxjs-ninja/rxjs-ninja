RxJS Primitives - String Operators

This library contains the RxJS Primitives for ECMAScript String methods.

Status

This library is currently in development. Please be aware APIs may change and there are missing features.

How to install

In your project run npm install @tinynodes/rxjs-string

To include in your project you can import the operators to include in any RxJS pipe

import { toUpperCase, toLowerCase } from '@tinynodes/rxjs-string';
import { BehaviorSubject, Observable } from 'rxjs';

export class TalkClass {
  $currentText = new BehaviorSubject<string>('');

  public shout(): Observable<string> {
    return this.$currentText.asObservable().pipe(toUpperCase());
  }
  public whisper(): Observable<string> {
    return this.$currentText.asObservable().pipe(toLowerCase());
  }
}

Available Methods

Method Operator Example
String.prototype.charAt charAt from(['Hello?']).pipe(charAt(2)).subscribe(// 'e')
String.prototype.charCodeAt charCodeAt from(['Hello?']).pipe(charCodeAt(2)).subscribe(// 101)
String.prototype.endsWith endsWith from(['Hello?']).pipe(endsWith('?')).subscribe(// true)
String.prototype.padEnd padString from(['1234']).pipe(padString('end', 5)).subscribe(// '1234 ')
String.prototype.padStart padString from(['1234']).pipe(padString('start', 5)).subscribe(// ' 1234')
String.prototype.toLocaleUpperCase toUpperCase from(['hello world']).pipe(toUpperCase()).subscribe(// 'HELLO WORLD')
String.prototype.toLocaleLowerCase toLowerCase from(['HELLO WORLD']).pipe(toLowerCase()).subscribe(// 'hello world')
String.prototype.trim trim from([' hello world ']).pipe(trim()).subscribe(// 'hello world')
String.prototype.trimStart trim from([' hello world ']).pipe(trim('start')).subscribe(// 'hello world ')
String.prototype.trimEnd trim from([' hello world ']).pipe(trim('end')).subscribe(// ' hello world')
