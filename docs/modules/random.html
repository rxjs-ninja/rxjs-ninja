    An Observable string value generator that generates random numbers using Math.random to generate a number between the min and max value passed. Pass an optional emit time to use as a timer

    example
    fromRandom(0, 10)
.pipe(tap(console.log))
.subscribe() // 8.332405921423248, 7.9304238877750315, 4.603376889484552...

    Parameters

    • Default value min: number = 0

      The minimum value to create a random number between

    • Default value max: number = 1

      The maximum value to create a random number between

    • Default value emitDelay: number = 0

      If set the observable will emit per millisecond set, by default this is 0

    Returns Observable<number>

  • fromRandomCrypto(emitDelay?: number, opts?: FromRandomCryptoOpts): Observable<number>
    • Defined in libs/rxjs/random/src/lib/from-random-crypto.ts:49

    An Observable string value generator that generates random numbers using Crypto.getRandomValues to generate a number. Pass an optional emit time to use as a timer.

    By default this will generate an unsigned 4-byte value Int32Array, pass additional options to instead generate a 1-byte or 2-byte signed or unsigned, or 4bit signed value.

    Typed Array Views

    example
    fromRandomCrypto()
.pipe(tap(console.log))
.subscribe() // 1228475997, 258463200, 850749141, 3060206219...
    example
    fromRandomCrypto(0, { bytes: 1 })
.pipe(tap(console.log))
.subscribe() // 15, 8, 234, 12...

    Parameters

    • Default value emitDelay: number = 0

      If set the observable will emit per millisecond set, by default this is 0

    • Default value opts: FromRandomCryptoOpts = DEFAULT_OPTIONS

      opts Options for different byte length of arrays, and unsigned and float values

    Returns Observable<number>

    Observable with a stream of random numbers

  • fromRandomStr(length?: number, emitDelay?: number, opts?: FromRandomStringOpts): Observable<string>
    • Defined in libs/rxjs/random/src/lib/from-random-string.ts:46

    An Observable string value generator that generates random strings of the passed length, if an emit delay is passed it will emit the value once per the time passed

    By default the string will be made up of upper and lower case characters, to include numbers and special characters pass a FromRandomStringOpts options object

    example
    fromRandomStr(5)
.pipe(tap(console.log))
.subscribe() // 'AgtYb', 'brYgT', 'TLAvf'...

    Parameters

    • Default value length: number = 10

      The length of the string to produce

    • Default value emitDelay: number = 0

      If set the observable will emit per millisecond set, by default this is 0

    • Default value opts: FromRandomStringOpts = DEFAULT_OPTIONS

      Additional options to include other characters and numbers

    Returns Observable<string>

    Observable with a stream of random strings
