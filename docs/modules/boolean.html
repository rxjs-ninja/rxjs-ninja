PredicateFn<T>: function
  • Defined in libs/rxjs/boolean/src/types/boolean.ts:34

A predicate function is used with filtering and should return a boolean based on an equality check.

Can be used for more complex conditions

example
const isEvenNumber: PredicateFn<number> = (num: number): boolean => num % 2 === 0
example
interface ObjectType {
  foo: number,
  bar: string
}
const isTheUltimateAnswer: PredicateFn<ObjectType> = (obj: ObjectType): boolean => obj.foo === 42
example
const nameIsAfter: PredicateFn<string, string> = (name1: string, name2: string): boolean => name1.localeCompare(name2) === 0
returns

Boolean value based on the condition of the function

  • fromBoolean(source: boolean | boolean[]): Observable<boolean>
    • Defined in libs/rxjs/boolean/src/lib/from-boolean.ts:37

    The fromBoolean operator is used to create an Observable boolean from a passed value or array of values. The operator can be overloaded with types to be used with strings, number or mixed sources

    Using fromBoolean with an array of booleans is the same as using the from operator from RxJS

    typeparam

    The type or types to be used to create boolean values from

    default

    boolean

    example
    fromBoolean(true)
 .pipe(map(val => !val))
 .subscribe(console.log) // false
    example
    fromBoolean([false, true, false, true])
 .pipe(map(val => !val))
 .subscribe(console.log) // [true, false, true, false]

    Parameters

    • source: boolean | boolean[]

      Boolean input to create an Observable from

    Returns Observable<boolean>

    Boolean value from the initial parameters
