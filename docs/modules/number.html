  • filterInRange(min: number, max: number, excludeBoundingValues?: undefined | false | true): MonoTypeOperatorFunction<number>
    • Defined in libs/rxjs/number/src/lib/filter-in-range.ts:44

    The filterInRange operator can be used with an RxJS pipe where the source value is an Observable number and will return the number value if the number is within a min/max value range. The optional excludeBoundingValues parameter allows the min and max values to be excluded.

    • If you want to get the boolean value of a number being outside a range of min/max use inRange operator instead
    • If you want the number value being outside a range of min/max use the filterInRange operator instead

    If excludeBoundingValues is set to true, the range will not include the actual min and max values by using > && < as it's equality match

    The default value is false and will use a >= && <= equality match

    example
    // Exluding the bounding number
fromNumber([-1, 0, 1, 2, 10, 11])
 .pipe(filterInRange(0, 10))
 .subscribe(console.log) // [0, 1, 2, 10]
    example
    fromNumber([-1, 0, 1, 2, 10, 11])
 .pipe(filterInRange(0, 10, true))
 .subscribe(console.log) // [1, 2]

    Parameters

    • min: number

      The minimum number for the range

    • max: number

      The maximum number for the range

    • Optional excludeBoundingValues: undefined | false | true

      If bounding values should be excluded

    Returns MonoTypeOperatorFunction<number>

    Number value if the number falls in the min/max range

  • fromNumber(source: number | number[]): Observable<number>
    • Defined in libs/rxjs/number/src/lib/from-number.ts:35

    The fromNumber operator is used to create an Observable using a number or array of numbers. The Observable can be subscribed to, which will emit one or more numbers.

    Using fromNumber with an array of numbers is functionally similar to the RxSJ from operator - each number in the array will emit a value to the subscription

    example
    fromNumber(6)
 .pipe(map(val => val * 7))
 .subscribe(console.log) // 42
    example
    fromNumber([1, 2, 3])
 .pipe(reduce((acc, val) => acc + val))
 .subscribe(console.log) // 6

    Parameters

    • source: number | number[]

      The number or array or numbers to start from

    Returns Observable<number>

    Observable created from the input array of numbers
