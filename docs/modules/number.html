RxJS Number Creation Functions

fromNumber

  • fromNumber(input: number | number[], scheduler?: SchedulerLike): Observable<number>
    • Defined in number/src/lib/from-number.ts:33

    The fromNumber operator is used to create an Observable number from a passed number value. A single number or array can be passed, if an array is passed each value is emitted

    example
    fromNumber(6 * 7)..subscribe(console.log) // 42
    example
    fromNumber([1, 2, 3])
 .pipe(reduce((acc, val) => acc + val))
 .subscribe(console.log) // 6
    remarks

    Using fromNumber with an array of numbers is the same as using the from operator from RxJS

    Parameters

    • input: number | number[]

      Number input to create an Observable from

    • Optional scheduler: SchedulerLike

      The SchedulerLike to use for scheduling the emission of values, and providing a notion of "time"

    Returns Observable<number>

    Observable number from passed input parameter
